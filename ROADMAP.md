# Roadmap

## Phase 1: Alpha (Current - v0.1.0-alpha.1)
**Goal:** Validate core design with early adopters

**Features:**
- ✅ Core message system
- ✅ 100+ essential messages
- ✅ JSON, Plain Text, XML,Log & Console formatters
- ✅ Parameter substitution
- ✅ Custom message loading

**What's Missing:**
- AspNetCore package
- Dependency Injection
- Built in Interceptors
- 200+ messages

**Timeline:** 2 weeks for feedback

---

## Phase 2: Beta (v0.2.0-beta.1)
**Target:** 2 weeks after alpha

**New Features:**
- AspNetCore package (.ToApiResponse())
- Logging integration
- Interceptor system and built in interceptors
- Improved and more formatters
- Expand to 200+ messages

**Focus:** ASP.NET Core integration

---

## Phase 3: Release Candidate (v1.0.0-rc.1)
**Target:** 4 weeks after alpha

**Focus:**
- Bug fixes only
- Documentation completion
- Performance testing

---

## Phase 4: Stable (v1.0.0)
**Target:** 6 weeks after alpha

**Commitment:**
- Production-ready
- Semantic versioning
- Long-term support

---
